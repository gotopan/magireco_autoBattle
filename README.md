# magireco_autoBattle
魔法纪录自动周回脚本，自动使用AP回复药，可以直接在手机端运行，国服台服通用  
基于autojs开发，可自行打包

## how to use

1. [下载最新apk](https://github.com/icegreentee/magireco_autoBattle/releases)，安装
2. 只需给APP悬浮窗权限
3. 每一次运行，需开启无障碍服务，设置需要的参数
4. 进入游戏选择助战阶段，悬浮窗中启动脚本
